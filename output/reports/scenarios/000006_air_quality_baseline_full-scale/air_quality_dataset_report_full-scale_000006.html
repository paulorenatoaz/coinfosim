CoInfoSim — UCI Air Quality Dataset Report

This report documents the fixed dataset preparation protocol. The five PT08 variables are metal-oxide sensor responses, not measured gas concentrations. The binary threshold is experimental and is not a health or regulatory limit.

1. Dataset identity and public provenance

Air Quality, UCI Machine Learning Repository dataset 360, DOI 10.24432/C59K5F. Official source: UCI Air Quality. UCI credits Saverio Vito and identifies the dataset as CC BY 4.0.

6. Benzene reference and binary target definition

C6H6(GT) is the certified-analyzer benzene reference and is excluded from classifier input X. The training-only linear 75th percentile is 14.5. Class 1 is defined by C6H6(GT) >= 14.5; class 0 is below that threshold.

17. Leakage-control notes

  • Threshold estimation uses training reference values only.
  • Standardization and Gaussian/GMM fitting use training rows only.
  • C6H6(GT), all other GT columns, T, RH, AH, Date, Time and timestamp are excluded from classifier inputs.
  • The fixed future real test is reused unchanged by Real → Real, Single Gaussian → Real and GMM → Real.

18. Limitations

The sensor array exhibits cross-sensitivity, sensor drift and concept drift. The chronological shift can affect empirical loss. Converting a continuous benzene reference into a binary target is an artificial binarization that discards magnitude. The threshold is relative to this training period and supports no compliance claim.
