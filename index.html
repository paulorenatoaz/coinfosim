CoInfoSim - Published Research Reports and Datasets

CoInfoSim is a research simulator for evaluating predictive cooperation across attribute subsets in supervised classification.

Install CoInfoSim

pip install coinfosim

Quick start

coinfosim scenario list
coinfosim scenario run occupancy --mode smoke

Available datasets

Files are mirrored byte-for-byte on this site; hashes are pinned in the installed package and verified before every run. See the machine-readable manifest.
